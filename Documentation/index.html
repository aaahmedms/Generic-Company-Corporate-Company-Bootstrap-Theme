The Generic Company web theme is a modular SPA style theme that is arranged in a vertical list format using a series of modular components, which comprise of containers and divs. The theme can be customized and organized however the user wishes by simply taking out a div, or container holding a section of the webpage, and then placing it somewhere else within the body of the webpage. The webpage also scales on mobile devices, and is supported on multiple resolutions.

This is the perfect web theme for anyone looking to create a professional website for their business, or portfolio and do not have the time to program it from scratch. The following documentation provided here will show the different ways to modify the webpage to better suit the users needs. But before reading further its advised to check the minimum requirements listed above, and to ensure some kind of text editor is used to modify each page.

A couple of notes: Images used on the theme are free for use and are licensed under CC0, however videos placed on the webpage are simply example videos which require the authors permission before using. The videos have simply been included as placeholder to show how to include videos from youtube, and should be replaced with your own. I do not take credit, or claim ownership for any of the videos used in the web theme, all rights for the videos used are held by the authors for their respective work.

The image to the right showcases the various sections that comprise the Generic Company web theme, all of which can be moved around or modified.

1. Getting Started #back to top

Because this is a complete beginners guide, the basics will be covered on how to get started.

Where to Host Web Theme for Others to See

If you do not have any type of hosting service to host your web page, then download a program called XAMPP to host the page locally to see how it will look on all browsers. Its important to note that the graphs will not show until the web page is hosted either on localhost, or on a server of some kind such as hostgator. An FTP client such as filezilla will streamline the process of uploading files onto your server wether it is hosted locally, or on a hosting service.

Download Link:

The link to download XAMPP for free is here: https://www.apachefriends.org/download.html

The link to download FileZilla for free is here: https://filezilla-project.org/download.php

Here is a good video guide on how to host locally using XAMPP:

If not hosting locally:

Simply ignore the above steps and use the guides provided by your hosting service.

Transferring Files Quickly to a Server:

Guides on how to quickly transfer files to any kind of server wether locally, or to a hosting service can be found on Youtube. Simply type in "FTP filezilla to server" on Youtube for a tutorial on how to do so. Using XAMPP to locally host does not require an FTP client as you can simply drag and drop files onto a localhost then test the web page quickly. For hosting services an FTP client is more useful as many services mainly accept a limited number of items at once, which becomes tedious and long to add each item one by one, so refer to an FTP client when using a hosting service.

2. Requirements #back to top

1. FileZilla - To transfer files quicker to a hosting service.

2. XAMPP - For local testing, to test out the web theme or test changes made to the web theme.

3. How to Setup Web Page: #back to top

Setting up the page is fairly simple. The first thing to do is simply extract the rar file included using an unzipping program such as winzip or winrar, then drag the files onto the folder of your choice.

From there you simply upload the folder onto a hosting service using FileZilla. If you would like to test the web page theme locally, then simply use XAMPP.

For FileZilla:

For FileZilla simply enter your login information for the service you are using, connect then drag and drop files.

For XAMPP:

For XAMPP run Apache and MySQL then press the "Explorer" button to access the localhost folder and upload the "WebPageTheme" folder in the htdocs folder.

You can now locally test the theme by typing in "http://localhost/WebPageTheme/" in any browser after the folder is placed in the "htdocs" folder.

Editing the Files:

The files within the folder (HTML,JS,CSS) can be edited via a text editor such as Notepad++.
